# Software creation of technical graphics

## Description

The textbook (in slovak language) is intended for those interested in creating professional graphics for publications in the field of electronics using [CircuitMacros](https://ece.uwaterloo.ca/~aplevich/Circuit_macros/). It describes basic procedures for drawing electronic circuit connections, methods for creating macros for creating custom components, and examples of using libraries. The source codes of the book are created in the Markdown language on the Sphinx platform.
