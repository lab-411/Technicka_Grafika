Funkcie #

Programovací jazyk dpic neumožňuje vytváranie nových uživateľských funkcií, sú v ňom ale definované základné matematické funkcie.

Trigonometrické funkcie #

Argumenty trigonometrických funkcií sú v radiánoch:

sin(<expr>)
cos(<expr>)
acos(<expr>)
asin(<expr>)
atan2(<expr>, <expr>)
tan(<expr>)

Logaritmické a exponenciálne funcie #

exp(<expr>)
log(<expr>)
expe(<expr>)
loge(<expr>)

Numerické funkcie #

sqrt(<expr>)
max(<expr>, <expr>...)
min(<expr>, <expr>...)
int(<expr>)
rand(<expr>)
abs(<expr>)
floor(<expr>)
sign(<expr>)
pmod(<expr>)

Príklady #

Nasledujúce príklady ukazujú použitie funkcií pri vykreslovaní jednoduchej grafiky, Obr. 70 a Obr. 71.

Grid(3,3);
    right_;
    r = 1.5;
    move to (0,r);
C1: circle diam 2*r;
    color_red;
    alpha = pi_ / 4;     # uhol v radianoch
    line from C1.c to C1.c + (cos(alpha), sin(alpha))*r;
D1: dot;

PP:(0,0)
for x=0 to 3.14*2 by (3.14/100) do {
    r1 = 2;
    px = cos(x);
    py = sin(2*x);

    r2 = 4;
    dx = r2*cos(1*x)/2 + 2;
    dy = r2*sin(3*x)/3 + 1;

    line from PP + (dx,dy)*r1 to PP + (px,py)*r2; 
}
